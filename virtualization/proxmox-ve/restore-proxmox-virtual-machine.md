# Restore Proxmox Virtual Machine

With the backup file in place, you can restore the virtual machine to its original state using the following simple steps. Head over to the backup drive and click on the backup file, then click the ‘**Restore**‘ button as shown.

<figure><img src="https://www.tecmint.com/wp-content/uploads/2024/01/Proxmox-VM-Restore.png" alt="Proxmox VM Restore" height="376" width="1384"><figcaption><p>Proxmox VM Restore</p></figcaption></figure>

On the pop-up dialogue box, click ‘**Restore**‘ to commence the VM restoration process.

<figure><img src="https://www.tecmint.com/wp-content/uploads/2024/01/Restore-Proxmox-VM.png" alt="Restore Proxmox VM" height="454" width="706"><figcaption><p>Restore Proxmox VM</p></figcaption></figure>

A task viewer will open detailing the restoration process as shown. This takes a few seconds to a minute, so be patient.

<figure><img src="https://www.tecmint.com/wp-content/uploads/2024/01/Proxmox-VM-Restore-Process.png" alt="Proxmox VM Restore Process" height="539" width="854"><figcaption><p>Proxmox VM Restore Process</p></figcaption></figure>

Once completed, you will see the ‘**TASK OK**‘ notification. Close the pop-up to exit.

<figure><img src="https://www.tecmint.com/wp-content/uploads/2024/01/Proxmox-VM-Restore-Completes.png" alt="Proxmox VM Restore Completes" height="543" width="861"><figcaption><p>Proxmox VM Restore Completes</p></figcaption></figure>

On the Proxmox dashboard, you should now see your restored virtual machine. To start using your VM, right-click and select ‘**Start**‘.

<figure><img src="https://www.tecmint.com/wp-content/uploads/2024/01/Proxmox-VM-Start.png" alt="Start VM in Proxmox" height="390" width="927"><figcaption><p>Start VM in Proxmox</p></figcaption></figure>
