# Reset Windows Server without Losing Data

**Steps to Reset Windows Server without Losing Data:**<br>

This video demonstrates how to access the advanced menu to reset your Windows PC:

1. **Enter Advanced Startup Menu:**
   * **Via Settings:** Go to **Settings** > **Update & Security** > **Recovery** > **Advanced startup** > **Restart now**.
   * **Via Sign-in Screen:** Hold down the `Shift` key, then select **Power** > **Restart**.
   * **Via Interruption:** Turn on the computer, and when the manufacturer logo appears, hold the power button until it shuts down. Repeat this process three times to trigger automatic repair, then select **Advanced options**.
2. **Navigate to Reset Option:** From the Choose an option screen, select **Troubleshoot**.
3. **Initiate Reset:** Select **Reset this PC**.
4. **Keep Data:** Choose the option **Keep my files**.
5. **Confirm:** Select **Local reinstall** (or Cloud download if needed). The system will show a summary of what will be done, then click **Reset** to start.&#x20;

**Key Considerations:**

* **Apps Removed:** While user files are safe, applications and customized settings will be removed, and a list of removed apps will be provided post-reset.
* **Backup:** Always perform a complete backup before initiating a reset to ensure data safety.
* **Alternative:** For a less intrusive fix, consider an "in-place upgrade," which can repair the system while keeping applications.&#x20;
